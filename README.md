Features posts, voting, user profiles, and live messaging — built on Django Channels with ASGI for async support.

`Django` `Django Channels` `WebSockets` `ASGI` `JavaScript`

---

### 🎬 [Entertainment Tracker](https://github.com/Pronoobcodes/Entertainment-Tracker)
> A personal media library that pulls live data from **TMDb, MyAnimeList & MangaDex**.

Track watch/read status across movies, anime, and manga — all in one place, with full user authentication.

`Django` `REST APIs` `TMDb API` `MyAnimeList API` `Bootstrap`

### 📂 [Download Organizer](https://github.com/Pronoobcodes/Download-Organizer)
> A **cross-platform file automation daemon** that keeps your Downloads folder sorted — automatically.

Monitors the filesystem in real time using Watchdog, sorts files by type, and auto-starts on boot — implemented in both Python and Java.

`Python` `Java` `Watchdog` `Automation` `Cross-platform`
